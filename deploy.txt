--- Deploying container to Azure ----------------------------

STEP 1 - PUBLUSH to Azure Container Registry (custom registry that you create in portal)

first login on CLUSTER_ID
    az login --tenant 3b3dbd39-f0c2-4af8-a422-9e388901f651

then login to your container registry

    az acr login --name mycontainerregistryname

We need to tag our container as below 

    docker tag visual-git-api mycontainerregistryname.azurecr.io/visual-git-api
    docker tag apache/kafka:3.7.1 mycontainerregistryname.azurecr.io/apache/kafka


provide access
    az acr update -n mycontainerregistryname --admin-enabled true  

And finally push to the container registry
    docker push mycontainerregistryname.azurecr.io/visual-git-api 
    docker push mycontainerregistryname.azurecr.io/mykafka
